# PhasmoSave
PhasmoSave integration for automatic PhasmoSave process!

### In order to run follow instructions.

> 1. Run  `Create.py`
> 2. Run  `Edit.py`
------
### How quick is the process? 
Edit.py will go through everything in file so does take about a minute to complete.
